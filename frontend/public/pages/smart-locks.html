¿Cómo Funciona?

Automatización completa en 4 pasos simples

1

🔗 Conecta tu Cerradura

Vincula tu smart lock existente o instala uno nuevo. Nuestro sistema detecta automáticamente la marca y modelo, configurando la conexión en menos de 2 minutos.

2

📅 Sincronización Automática

Cada nueva reserva activa automáticamente la generación de códigos. El sistema calcula las fechas y crea códigos únicos para cada huésped.

3

📱 Envío Automático

2 horas antes del check-in, el huésped recibe automáticamente el código por WhatsApp junto con las instrucciones de acceso.

4

🔒 Expiración Automática

El código expira automáticamente a la hora de checkout. El sistema genera un nuevo código para la siguiente reserva.

Funcionalidades Avanzadas

Códigos Temporales

Cada código tiene fecha y hora de activación/expiración exactas. Se activan 2 horas antes del check-in y expiran automáticamente en el checkout.

📊

Historial Completo

Registro detallado de todos los accesos: quién, cuándo y desde dónde. Exportable para seguros o reclamaciones.

🚨

Alertas Inteligentes

Notificaciones instantáneas de accesos no autorizados, intentos fallidos o manipulación de la cerradura.

🔄

Backup Automático

Si falla la conexión principal, el sistema activa automáticamente códigos de emergencia y notifica al host.

📱

Control Remoto Total

Abre/cierra remotamente, genera códigos de emergencia para limpieza o mantenimiento desde cualquier lugar.

🔋

Monitoreo de Batería

Alertas automáticas cuando la batería baja del 20%. Tiempo estimado de reemplazo y recordatorios proactivos.
